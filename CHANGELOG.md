# Changelog

## 0.2.3 - 2026-02-14

### Added

- `--quick-config` to apply and persist a practical baseline profile in one command.
- `--enhance git` to enable detailed Git visibility.
- `--enhance observability` to enable usage, limits, session, and version visibility.

### Changed

- `npx codexline` now opens interactive mode by default in TTY terminals.
- Non-interactive contexts (for example CI and pipes) still default to `--plain`.

### Docs

- Updated command examples and behavior notes for quick profile and enhancement flows.
